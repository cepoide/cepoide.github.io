Configurar Token de Acceso

Para usar el panel, necesitas un Token de Acceso Personal de GitHub.

  1. Ve a GitHub para generar un nuevo token (clásico).
  2. Dale una nota (ej: "Admin Panel Sitio").
  3. Selecciona una fecha de expiración.
  4. Marca la casilla de permisos `repo`.
  5. Copia el token generado (empieza con `ghp_...`).
  6. Pégalo en el campo de abajo y guárdalo.
